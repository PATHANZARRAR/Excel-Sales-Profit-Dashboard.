# Sales & Profit Analytics Dashboard (MS Excel)

## Project Overview
This project showcases a professional-grade Sales and Profit Dashboard built entirely in MS Excel 2007. It transforms raw sales data into actionable business insights.

## Features & Highlights
* **Data Visualization:** Uses Column Charts and Pie Charts for easy trend analysis.
* **Interactive Reporting:** Implemented 'Report Filters' for region-wise data drilling.
* **Financial Formatting:** Integrated currency symbols and professional accounting formats.
* **Clean UI:** Removed gridlines and optimized chart alignment for a software-like interface.

## Key KPIs Tracked
* Total Sales by Region
* Profitability by Product Category
* Order Quantity distribution

## How it Works
1. Data is maintained in the 'Data' sheet.
2. Pivot Tables summarize the data on a hidden/separate sheet.
3. The 'Dashboard' sheet fetches this data to update visuals automatically.
